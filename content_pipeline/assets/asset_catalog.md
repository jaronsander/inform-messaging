## Usage Guidelines

### When Generating Content:
1. Read this catalog and search for relevant assets
2. Consider the emotion/theme of your content
3. Suggest 1-2 Roy poses that match the content tone
4. Suggest any supporting elements if available
5. Only request new asset generation if no match exists

### When Generating Assets:
1. Follow the frozen prompt from `knowledge/brand/roy_profile.md`
2. Maintain transparent backgrounds for all assets
3. Always include bold white border (die-cut sticker effect)
4. Generate both with/without toolbox variations for Roy poses
5. Update this catalog immediately after generation with full details

### Asset Naming Convention:
- Roy poses: `roy_[emotion]_[key_descriptor].png`
- Elements: `element_[object]_[state].png`
- Backgrounds: `bg_[description].png`
